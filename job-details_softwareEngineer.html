Role information

  • Category : Software Engineer
  • Job Description :
  • Analyze, design, develop, and maintain various software applications.

  • Work with Business Analysts and Product Owners to identify potential issues.

  • Implement Agile methodology, create and prioritize user stories.

  • Develop, maintain micro-services, and create business layer components using the spring-based framework.

  • Design and implement web applications using Java technologies and write test cases using JUnit and Mockito framework.

  • Use data structures, and algorithms and optimize code performance.

  • Operate Angular and HTML to develop UI screens and take part in the Software Development Life Cycle.

  • Deploy code in DEV and Test regions using Jenkins and update confluence pages as per project needs.

  • Perform CRUD operations on files and folders in AWS S3 buckets.

  • Utilize AWS ECS services to monitor deployed services in the DEV region.

  • Participate in brainstorming, sprint planning, code review sessions, ideathons to generate ideas and solutions.

  • Use JIRA, and Clear Quest for tracking projects and activities.

  • Handle release management activities during production go live.

  • Required tools and utilities: :
  • MySQL, AWS-S3, Cloud Watch, EC2, Splunk Enterprise & Postman. 

  • Other:

  • Full Time.

  • Travel involved to unanticipated client locations within the US.

  • Requirements:

  • Master's in Computer and Information Science or Equivalent.

  • Any suitable combination of education, training, or experience is accepted. 

  • Location:

  • Suwanee, GA.
